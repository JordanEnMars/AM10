Problem 1: Falling object with position-dependent drag¶

In this problem you will compute position, velocity, and accelaration of an object falling under gravity with frictional drag (similar to the exercises done in lecture and section).

The force on the mass is given by $$F(t) = -mg + \frac{1}{2} \gamma(x) [v(t)]^2,$$ as you have seen before. This time the drag coefficient, $\gamma$ is position-dependent with the following form.

$\gamma(x) = \gamma_0e^{-kx(t)}$.

Let's think of what this means. While $x>>1$ the drag is nearly zero. As $x$ approaches $0$ the drag approaches the value $\gamma_0$. As $x$ becomes negative, the drag increases exponentially. Also, note that larger the value of $k$, steeper is the exponential increase. So, this position dependent drag coefficinet is like a simple toy model of an object falling through a material interface at $x=0$, i.e., falling from one medium with low drag (say air) into another with very high drag (say oil).

Feel free to copy code from the lecture and/or section and modify appropriately. Use $\gamma0=1$ and $k=0.5$.

Part A: Produce clearly labeled plots of the position, velocity, and accelaration of the falling object. Use a for loop to run the time propagation for a maximum time of 10 units with 1000 time steps.

\begin{equation} C(t) = \left [ r(1-p)t + C_0^{1-p} \right ]^{\frac{1}{1-p}}, \end{equation}

where $C(t)$ is the cumulative number of cases as a function of time, $C_0$ is the number of cases at $t=0$, $r$ denotes the growth rate, $p$ is the "deceleration of growth" parameter.

data=np.loadtxt("AM10-HW-05-US-cases-timeseries.txt")
time=data[:,0]
cases=data[:,1]
dx=1
def central_diff(cases, dx):   
    nx = len(cases)
    d1_cent=np.zeros(nx)
    for n in range (1, nx-1):
        d1_cent[n] = (cases[n+1] - cases[n-1]) / (2.0 * dx)
            
    # Improve end points by first-order extrapolation
    d1_cent[0] = 2 * d1_cent[1] - d1_cent[2]
    d1_cent[nx - 1] = 2 * d1_cent[nx - 2] - d1_cent[nx - 3]

    return d1_cent
numerical_deriv=central_diff(cases,dx)

# Part A
def three_point_diff(fx, dx): 
    nx = len(fx)
    d2fx=np.zeros(nx)
    for n in range (1, nx-1):
        d2fx[n] = (fx[n+1]-(2*fx[n]) + fx[n-1]) / ((dx)**2)
            
    # Improve end points by first-order extrapolation
    d2fx[0] = 2 * d2fx[1] - d2fx[2]
    d2fx[nx - 1] = 2 * d2fx[nx - 2] - d2fx[nx - 3]

    return d2fx
